Contact Form with Validation

An interactive contact form with real-time validation, error handling, and user feedback

HTML5 CSS3 JavaScript Form Validation Error Handling

Project Overview

A contact form with client-side validation to ensure data quality before submission, providing clear feedback to users.

Key Features

Real-time Validation

Validates inputs as user types with immediate feedback for errors.

Email Validation

Regex-based email format checking to ensure valid email addresses.

Required Fields

Ensures all required fields are filled before form submission.

Error Messages

Clear, user-friendly error messages displayed below each field.

Success Feedback

Visual confirmation when form is successfully submitted.

Accessible

Proper labels, ARIA attributes, and keyboard navigation support.

Code Sample

Example of the form validation script:

// Email validation function function validateEmail(email) { const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; return regex.test(email); } // Form submission handler form.addEventListener('submit', (e) => { e.preventDefault(); let isValid = true; // Validate name if (nameInput.value.trim() === '') { showError(nameInput, 'Name is required'); isValid = false; } // Validate email if (!validateEmail(emailInput.value)) { showError(emailInput, 'Please enter a valid email'); isValid = false; } if (isValid) { showSuccess('Form submitted successfully!'); } });
