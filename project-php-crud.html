PHP CRUD Mini System

A simple backend system with forms and records (create/read/update/delete) to practice PHP basics

PHP MySQL CRUD Operations HTML Forms Database

Project Overview

A mini content management system built with PHP and MySQL to practice backend development fundamentals.

Key Features

Create Records

Add new data entries through HTML forms with validation.

Read Records

Display all records from database in a table format.

Update Records

Edit existing data entries with pre-filled forms.

Delete Records

Remove records from database with confirmation.

Database Connection

PDO-based MySQL connection for secure data handling.

Form Validation

Basic server-side validation for data integrity.

Code Sample

Example of the create operation:

// Insert new record into database if ($_SERVER['REQUEST_METHOD'] === 'POST') { $name = $_POST['name']; $email = $_POST['email']; $sql = "INSERT INTO users (name, email) VALUES (:name, :email)"; $stmt = $pdo->prepare($sql); $stmt->execute([ ':name' => $name, ':email' => $email ]); header('Location: index.php'); }

What I Learned

  • PHP form handling and POST requests
  • MySQL database operations with PDO
  • CRUD (Create, Read, Update, Delete) patterns
  • Basic security practices (prepared statements)
